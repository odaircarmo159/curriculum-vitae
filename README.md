# Currículo Vitae

Projeto em React + Vite com uma página única de currículo/portfólio, em português e sem dependências desnecessárias do template original.

## Rodando localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## O que ajustar no conteúdo

- Atualizar e-mail, telefone, LinkedIn e localização em `src/App.tsx`
- Substituir experiências pelos cargos e empresas reais
- Ajustar projetos e competências conforme seu histórico
